# HTML Resume Website

A multi-page resume website built with semantic HTML and deployed on GitHub Pages 
for CIS 4004 at UCF.

## Pages

**Main Resume Page (index.html)**
- Full name, UCF email, and LinkedIn profile link
- Professional objective statement
- Education
- Relevant work experience
- Projects
- Skills

**Project Pages**
- Two additional pages each dedicated to a unique project
- Each project page includes a title, illustrative image, project description, 
  and skills learned

## Requirements Met
- Deployed and live on GitHub Pages
- Semantic HTML tags used throughout (header, nav, main, section, article, footer, etc.)
- Non-semantic tags minimized
- All required resume sections present
- Two fully built-out project detail pages

## How to View
Visit the live site: [samiratamer.github.io](https://samiratamer.github.io)

Or open `index.html` locally in any web browser — no setup required.

## Concepts Used
- Semantic HTML5 structure
- Multi-page website architecture
- Hyperlinks and navigation between pages
- HTML lists, tables, and text formatting
- GitHub Pages deployment
